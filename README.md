# ebbie

ebbie は、小規模Webサイト開発で利用する事を想定した SASS(SCSS)/gulp のパッケージです。

## Installation

[Node.js](https://nodejs.org/en/) をインストールして下さい。

次に以下のコマンドを入力して下さい。

```
$ npm install -g gulp
$ cd path/to/ebbie
$ npm install
$ gulp
```
